# PLG-READER

**PLG-READER** - это токенизатор (лексер) для компилятора [py2glua](https://github.com/py-love-gmod/py2glua).
Он читает файлы разных типов и превращает их в единый список IR-токенов.

Библиотека не выполняет код.

## Зачем нужна эта библиотека?
Чтобы компилятор мог единообразно читать все четыре типа файлов и сразу получать IR с правильными метками.

## Поддерживаемые типы файлов

| Расширение | Что это на самом деле |
|------------|-----------------------|
| `.py`      | Обычный Python-код    |
| `.pyi`     | Файл-описатель (stub) |
| `.plg-d`   | Алиас на внешний аддон (динамическая ссылка)   |
| `.plg-s`   | Код, компилируемый внутрь аддона (статический) |
