# PLG-READER

**PLG-READER** - это парсер и конвертор в IR для компилятора [py2glua](https://github.com/py-love-gmod/py2glua).
Он читает файлы разных типов и превращает их в единый список IR-токенов.

Библиотека не выполняет код который парсит.

## Зачем нужна эта библиотека?
Чтобы компилятор мог единообразно читать все четыре типа файлов и сразу получать IR с правильными метками.

## Поддерживаемые типы файлов

| Расширение | Что это на самом деле    |
|------------|--------------------------|
| `.py`      | Обычный Python-код       |
| `.pyi`     | Файл-описатель (stub)    |

## Бинарный дампер

Бинарный дампер `BinaryRW` для стандартных типов python

## CLI утилита

[clii](https://github.com/jamesob/clii) by James O'Beirne - микро-обёртка над argparse (встроена в пакет сразу, немного изменена чтобы покрыть свои нужды)
